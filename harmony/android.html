Harmony for Android

  1. On your Android device, open this website, and download the latest .apk file from the releases tab (here).
  2. Once downloaded, go into the Files app (works with any files app), go to your Downloads folder, then tap on the file you just downloaded.
  3. If you haven't installed an apk with the files app, it should prompt you to allow the app to install 'Unknown Apps' - toggle that on, and Google Play may prompt you to scan the file. Just accept as the app is safe and open-source.
  4. It will now prompt you and ask if you want to install or update Harmony. Tap Install.
  5. It's then installed, and you can enjoy!

*Please note that an internet connection is required to access all Harmony services.
